# mail Authentication Lab – Postfix, OpenDKIM and DNS Configuration

## Overview

This project documents the deployment of a Linux-based mail server using Postfix and OpenDKIM on Ubuntu 24.04 LTS running on a Kamatera VPS.

## Features

SMTP server configuration with Postfix
DKIM email signing with OpenDKIM
DNS-based email authentication
Linux server administration
Email deliverability improvements

## Technologies Used

Ubuntu 24.04 LTS
Postfix
OpenDKIM
DNS
SMTP
Kamatera VPS

## Project Structure

postfix/ - Postfix configuration files
opendkim/ - OpenDKIM configuration and public DKIM record
screenshots/ - Testing and validation screenshots

## Outcome

Successfully configured DKIM signing for outbound email and validated authenticated email delivery.
